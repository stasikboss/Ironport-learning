## 🔧 Alternative: Manual Deployment

If you prefer manual control:

### **Option A: GitHub + Vercel Website**

1. **GitHub:**
   - Go to https://github.com/new
   - Create repo: `ironport-learning`
   - Run: `PUSH-TO-GITHUB.bat`

2. **Vercel:**
   - Go to https://vercel.com/new
   - Click "Import Git Repository"
   - Select `ironport-learning`
   - Click "Deploy"

### **Option B: Command Line**

```bash
# 1. Authenticate
gh auth login
vercel login

# 2. Deploy
gh repo create ironport-learning --private --source=. --push
vercel --prod
```
